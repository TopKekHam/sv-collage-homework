Exercise 9

Compiler

Compliler is a program that takes an input (source code of the program) and produces instractions for a machine to execute.
The machine can be some sort of processor (CPU/GPU), or a JIT program like .NET runtime that reads the intermediate
language byte code that the C#/F#/VBasic compilers produced and converts its to the native machine byte code for
the processor to execute.

Interpreter

Interpreter is a program that runs an input as a program
(like python) and does not produce an executable to run.

Extra

In past years the difference between compiler and interpreter getting smaller, because the interpreters starting to create
an machine code to run without creating an executable like V8.
Another example, JAI compiler can run the source code without compiling (JAI is parallel to C in a type of usage).



Exercise 10

Math.sign(num : number) : number

This method returns 1 or -1 depends on the sign bit of the number.
If the number is 0 or -0 it returns it as is.
If param is Infinity or -Infinity works as well.
If param is NaN returns Nan.
If param is any data type that not a number returns NaN.

Math.abs(num : number) : number

This method returns absolute of a number, for exaple, if we pass -5 to the method we get 5 as a return value.
If param is Infinity or -Infinity works as well.
If param is NaN returns NaN.
If param is any data type that not a number returns NaN.
